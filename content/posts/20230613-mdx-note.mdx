## codeblock Syntax Highlighting style is prism-tomorrow
addtional command including
- show line number
  ```tsx:file.tsx showLineNumbers
  function MyApp({ Component, pageProps }: AppProps) {
    return (
      <ThemeProvider attribute="class">
        <Head>
          <meta name="viewport" content="viewport-fit=cover" />
        </Head>
        <LayoutWrapper>
          <Component {...pageProps} />
        </LayoutWrapper>
      </ThemeProvider>
    );
  }
  ```
- highlight specific code segments
  ```tsx:file.tsx showLineNumbers {2,4-5}
  function MyApp({ Component, pageProps }: AppProps) {
    return (
      <ThemeProvider attribute="class">
        <Head>
          <meta name="viewport" content="viewport-fit=cover" />
        </Head>
        <LayoutWrapper>
          <Component {...pageProps} />
        </LayoutWrapper>
      </ThemeProvider>
    );
  }
  ```
- show difference
    ```diff-js showLineNumbers {3-4}
    const a = 1;
    - a = 2;
    + a = 2;
    a = 3;
    ```
- show title
    ```python:Hello.py
    print("Hello world!")
    ```
